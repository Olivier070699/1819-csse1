Wat is het verschil tussen front end back end? 
-	Front end bestaat voornamelijk uit HTML, Css & javascript dit draait allemaal in de browser en zorgt zo voor de graphical user interface. Back end bestaat uit bv java, C#, Phython, PHP, SQL, … deze code draait op de server en is onzichtbaar voor de gebruiker.
